## Decisiones y aprendizajes clave

- **Vercel Analytics + eventos propios** reemplazó la idea de un contador de visitas público — riesgo de exponer números bajos ante usuarios/inversionistas.
- **Regla dura para contenido IA:** nunca inventar estadísticas, citas o atribuciones. El incidente de los artículos reforzó que el texto debe copiarse literal desde los .md aprobados, no "mejorarse" en el camino.
- **El magic link en mobile no es un bug** — es comportamiento estándar del mini-navegador de Gmail en Android. La sesión se mantiene aunque la ventana se cierre. Se resolvió con mensajes de claridad, no con solución técnica.
- **Deploy manual con Vercel CLI** (`npx vercel --prod`) como workaround mientras el webhook estaba roto. El auto-deploy ya funciona, pero el webhook no aparece en GitHub Settings → Webhooks (Vercel usa GitHub App, no webhook clásico).
