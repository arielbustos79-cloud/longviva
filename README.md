# LongViva 🌿

**Tu prime, tu plataforma.**

LongViva es una plataforma digital 100% gratuita de salud, bienestar y experiencias para personas en su prime (+60 años) en Chile.

## ¿Qué es LongViva?

Una plataforma donde la vitalidad, la plenitud y el protagonismo son el centro. Nos financiamos con publicidad segmentada y alianzas B2B — nunca con cuotas de los usuarios.

## ¿Qué incluye?

- 💬 **VIVIAN** — Asistente IA conversacional, cálida y directa
- 📅 **Agenda** — Citas médicas, clases y recordatorio de medicamentos
- 🏥 **Salud** — Telemedicina y orientación en salud
- 🤝 **Comunidad** — Grupos, actividades y conexión real
- 🌎 **Experiencias** — Tours, talleres y planes para vivir el presente

## Stack tecnológico

- **Frontend:** Next.js 15 (App Router) + Tailwind CSS
- **IA:** Claude API (Anthropic) — modelo claude-sonnet-4-6
- **Base de datos:** Supabase (PostgreSQL)
- **Auth:** Supabase Auth (magic link — sin contraseña)
- **App móvil:** React Native + Expo
- **Hosting:** Vercel

## Inicio rápido

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
